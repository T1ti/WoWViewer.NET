## WoWViewer.NET
This exists purely as an educational project for me to learn more 3D programming as well as how World of Warcraft renders its world. 

No support is given or to be expected when using building or using this project.

### Building
This requires the [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0) SDK to build.

### Running
This requires the [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0) Runtime to run.
